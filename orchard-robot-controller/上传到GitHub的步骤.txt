========================================
方案 A：Codemagic 在线构建 - 详细步骤
========================================

由于你的系统没有安装 Git，使用网页上传更简单！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 1 步：创建 GitHub 仓库
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 访问：https://github.com
2. 如果没有账号，点击 "Sign up" 注册（免费）
3. 登录后，点击右上角 "+" → "New repository"
4. 填写信息：
   - Repository name: orchard-robot-controller
   - Description: 果园机器人安卓控制器
   - 选择 Public（公开）
   - ✅ 勾选 "Add a README file"
5. 点击 "Create repository"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 2 步：上传项目文件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法 1：网页上传（推荐）

1. 在仓库页面，点击 "Add file" → "Upload files"
2. 将整个项目文件夹拖拽到网页（或点击选择文件）
3. 等待上传完成
4. 在底部填写：
   - Commit message: "Initial commit"
5. 点击 "Commit changes"

⚠️ 注意：一次最多上传 100 个文件，如果文件太多，分批上传：
   - 第一批：lib 文件夹
   - 第二批：android 文件夹
   - 第三批：其他文件

方法 2：使用 GitHub Desktop（更简单）

1. 下载 GitHub Desktop：https://desktop.github.com
2. 安装并登录
3. File → Add Local Repository → 选择项目文件夹
4. 点击 "Publish repository"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 3 步：配置 Codemagic
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 访问：https://codemagic.io/signup
2. 点击 "Sign up with GitHub"
3. 授权 Codemagic 访问你的 GitHub

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 4 步：添加项目到 Codemagic
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 在 Codemagic 控制台，点击 "Add application"
2. 选择 "GitHub"
3. 找到 "orchard-robot-controller" 仓库
4. 点击仓库名称
5. 点击 "Finish: Add application"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 5 步：开始构建
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Codemagic 会自动检测到配置文件
2. 点击 "Start new build"
3. 选择 "android-workflow"
4. 点击 "Start new build" 确认

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 6 步：等待构建（5-10 分钟）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

你会看到实时日志：
✓ Preparing build machine
✓ Cloning repository
✓ Installing Flutter
✓ Getting packages
✓ Building APK
✓ Build successful!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 7 步：下载 APK
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 构建完成后，滚动到页面底部
2. 在 "Artifacts" 部分
3. 点击下载 "app-release.apk"（约 20-30 MB）

推荐下载：app-arm64-v8a-release.apk（体积更小）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第 8 步：安装到手机
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法 1：USB 传输
1. 用数据线连接手机和电脑
2. 将 APK 复制到手机
3. 在手机上找到 APK 文件
4. 点击安装

方法 2：微信/QQ 传输
1. 将 APK 发送到"文件传输助手"
2. 在手机上下载
3. 点击安装

方法 3：扫码下载
1. 在 Codemagic 构建页面有二维码
2. 手机扫码直接下载

⚠️ 安装时需要：
- 允许"安装未知来源应用"
- 允许网络权限

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
完成！
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

应用安装后：
1. 打开应用（横屏显示）
2. 配置 ROS 服务器地址
3. 连接机器人
4. 开始控制！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
遇到问题？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. GitHub 上传失败 → 使用 GitHub Desktop
2. Codemagic 构建失败 → 查看日志，通常会自动修复
3. APK 无法安装 → 检查手机安全设置
4. 应用闪退 → 检查网络权限
